Detailed Description

attribute for modeleditor keys

This class implement model editor attribute to work with the modelEditor command, this attribute can support all the bool-argument of the maya command cmds.modeleditor For now only a small subset of those attribute has been tested and supported Accepted values for this attribute are BOOL only means True or False only
